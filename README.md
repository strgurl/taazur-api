# Taazur – Team Formation System

## Overview

Taazur is a hybrid team formation system that recommends suitable teammates based on skill complementarity. It combines a Gradient Boosting model with rule-based filtering to generate balanced and practical team recommendations.

## Features

* Complementarity-based matching 
* Hybrid approach: Machine Learning + Rule-Based filtering
* Explainable recommendations
* Web-based interface

## Tech Stack

Python, Flask, Scikit-learn, HTML/CSS
